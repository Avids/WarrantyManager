Form Panel — slide-in form to enter all fields from the Guild Electric certificate: project name/contract no., address, issued to, subject, dates, warranty period (1/2/3/5 years or custom), W.O. #, PM, and notes.
Warranty List — cards split into three live sections:

🟢 Active — with days remaining
🟡 Expiring — highlighted amber when ≤30 days left
⬛ Expired — grayed out, showing how many days ago

PDF Generation — clicking PDF on any card generates and downloads a properly formatted Guild Electric Limited Warranty Certificate matching your template exactly — with header, logo, fields table, all body sections, signature block, and footer.
Save / Open — exports/imports the full warranty database as a .json file so you can maintain records across sessions.
Search & Filter — live search across all fields, plus tab filters for All / Active / Expiring / Expired.
